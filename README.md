# java-racingcar-precourse

## 프로세스

- 자동차 이름 입력 - 시도 횟수 입력 - 게임 실행 - 차수별 결과 출력 - 우승자 출력

## 기능 목록

### 입력

- [ ] 자동차 이름 입력
- [ ] 시도 횟수 입력
- [ ] 이름 쉼표로 구분

### 게임

- [ ] 자동차 전진 구현
- [ ] 랜덤 숫자 생성 (0~9)
- [ ] 차수별 결과 반환
- [ ] 우승자 반환

### 출력

- [ ] 차수별 결과 출력
- [ ] 우승자 출력

### 예외

- [ ] 이름 5자 이하만 가능
- [ ] 시도 횟수는 자연수만 가능
